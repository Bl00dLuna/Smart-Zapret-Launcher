## ✨ Преимущества

### 🎯 Умная система приоритетов
**Автоматическое распределение трафика** • **Исключение конфликтов** • **Приоритет: Discord → YouTube/Twitch → Gaming → Universal**

### 🛠️ Профессиональная организация
**30+ готовых конфигов** • **Специализированные настройки** • **Гибкая комбинация**

### 🔧 Удобство использования
**Понятный интерфейс** • **Автосохранение настроек** • **Широкая совместимость** • **Автозапуск**

## 📖 Типы конфигов

| Сервис | Возможности |
|--------|-------------|
| **🔊 Discord** | Общайтесь, делитесь файлами, пользуйтесь демонстрацией экрана |
| **📺 YouTube/Twitch** | Воспроизведение видео, стабильные стримы, шортсы и клипы |
| **🎮 Gaming** | Играйте в любимые игры, несмотря на блокировки |
| **🌐 Universal** | Пользуйтесь сайтами как раньше |

## 🚀 Быстрый старт

1. Скачайте [последний релиз](https://github.com/Bl00dLuna/Smart-Zapret-Launcher/releases) и распакуйте архив
2. Запустите `Smart_Zapret_Launcher.bat` от имени администратора ✅
3. Выберите нужные конфиги в меню
4. Наслаждайтесь свободным интернетом! 🎉

## 🔥 Что-то не работает?

Если у вас возникли проблемы:

1. Прочитайте инструкции в архиве
2. Проверьте наличие прав администратора
3. Убедитесь, что антивирус не блокирует программу
4. Попробуйте разные конфиги
5. Обратитесь к уже созданным тикетам в Issues на GitHub - возможно, решение вашей проблемы уже есть
6. Если решения нет, создайте новый тикет
